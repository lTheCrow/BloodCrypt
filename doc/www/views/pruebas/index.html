Reto 1: Hola Mundo

Crear un programa que muestre algun texto en consola.

  1. Crear un archivo llamado hello.c donde se escribira el programa
  2. Compilar el archivo ya sea usando el GCC de MinGW o CL (el compilador de Microsoft que viene con Visual Studio)
  3. Enlazar el archivo y que el nombre del output sea hello.exe

Reto 2: Conversion de Temperatura

Crear un programa que convierta el valor numerico en Fahrenheit dado por un usuario a Celsius

El programa debe recibir una entrada por teclado con un numero ya sea entero o flotante

La entrada puede ser de tipo entero o flotante pero la salida solo puede ser de flotante y con 3 decimales despues de la coma

No les dare la formula, eso lo van a tener que investigar ustedes :)

Ejemplo:

Input: 49.319
Output: 9.622

Input: 69
Output: 20.5556
  1. Crear un archivo llamado temp.c donde se escribira el programa
  2. Compilar el archivo ya sea usando el GCC de MinGW o CL (el compilador de Microsoft que viene con Visual Studio)
  3. Enlazar el archivo y que el nombre del output sea temp.exe

Reto 3: Registro de Usuarios

Crear un programa que tenga un prompt ($>) que muestre dos opciones, si iniciar sesion, registrarse o salir

Al registrar un usuario, los datos se guardaran en un archivo, este archivo despues debe ser usado para iniciar sesion

Cualquier detalle agregado sera valorado ;)

El siguiente es un ejemplo que muestra todas las caracteristicas del programa:

> superlogin.exe
--- SuperLogin ---
1. Iniciar Sesion
2. Registrarse
3. Salir
$> 2
Nombre de Usuario: JohnDoe69
Clave: 1234
[+] Usuario Registrado !
--- SuperLogin ---
1. Iniciar Sesion
2. Registrarse
3. Salir
$> 3
> superlogin.exe
--- SuperLogin ---
1. Iniciar Sesion
2. Registrarse
3. Salir
$> 1
Nombre de Usuario: JohnDoe69
Clave: 4321
[+] Datos Incorrectos !
--- SuperLogin ---
1. Iniciar Sesion
2. Registrarse
3. Salir
$> 1
Nombre de Usuario: JohnDoe69
Clave: 1234
[+] Los datos son correctos, iniciando sesion --- Bienvenido al sistema JohnDoe69 --- >
  1. Crear un archivo llamado superlogin.c donde se escribira el programa
  2. Compilar el archivo ya sea usando el GCC de MinGW o CL (el compilador de Microsoft que viene con Visual Studio)
  3. Enlazar el archivo y que el nombre del output sea superlogin.exe
